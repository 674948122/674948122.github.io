# 在package.json加入命令

"scripts": {
    "lint": "eslint --ext .js,.vue src",
    "lint:fix": "eslint --fix --ext .js,.vue src",
 "lint:report": "eslint . --ext .js --format=html -o eslint-report.html",
  }

命令说明：

# npm run lint

校验语法规范，并输出错误

# npm run lint:fix

校验语法规范、输出错误，并进行一些基本修复

# npm run lint:report

校验语法规范、输出报告文档，命令运行完毕会在根目录下创建文件eslint-report.html，直接在浏览器中打开即可观看。
